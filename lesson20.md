**requests** ‚Äî —ç—Ç–æ —Å—Ç–æ—Ä–æ–Ω–Ω—è—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å `HTTP-–∑–∞–ø—Ä–æ—Å—ã` (`GET`, `POST`, `PUT`, `DELETE` –∏ –¥—Ä.) –∏ –ø–æ–ª—É—á–∞—Ç—å –æ—Ç–≤–µ—Ç—ã –æ—Ç –≤–µ–±-—Å–µ—Ä–≤–µ—Ä–æ–≤. –ë–µ–∑ —ç—Ç–æ–π –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ —Ä–∞–±–æ—Ç–∞—Ç—å —Å –∏–Ω—Ç–µ—Ä–Ω–µ—Ç–æ–º –±—ã–ª–æ –±—ã –∫—Ä–∞–π–Ω–µ –Ω–µ—É–¥–æ–±–Ω–æ ‚Äî –ø—Ä–∏—Ö–æ–¥–∏–ª–æ—Å—å –±—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –º–æ–¥—É–ª–∏ `urllib`, –∫–æ—Ç–æ—Ä—ã–µ –≥–æ—Ä–∞–∑–¥–æ —Å–ª–æ–∂–Ω–µ–µ –≤ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏.

### –û—Ç–ø—Ä–∞–≤–∫–∞ GET-–∑–∞–ø—Ä–æ—Å–∞


- `requests.get(...)` ‚Äî —ç—Ç–æ –∫–æ–º–∞–Ω–¥–∞, –∫–æ—Ç–æ—Ä–∞—è –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç GET-–∑–∞–ø—Ä–æ—Å –Ω–∞ —É–∫–∞–∑–∞–Ω–Ω—ã–π URL. –í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ ‚Äî –Ω–∞ GitHub API.

- `response.headers` ‚Äî —ç—Ç–æ —Å–ª–æ–≤–∞—Ä—å —Å HTTP-–∑–∞–≥–æ–ª–æ–≤–∫–∞–º–∏ –æ—Ç–≤–µ—Ç–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ç–∏–ø –∫–æ–Ω—Ç–µ–Ω—Ç–∞, –¥–∞—Ç–∞, –¥–ª–∏–Ω–∞).

- `response.json()` ‚Äî –µ—Å–ª–∏ —Å–µ—Ä–≤–µ—Ä –ø—Ä–∏—Å–ª–∞–ª –¥–∞–Ω–Ω—ã–µ –≤ —Ñ–æ—Ä–º–∞—Ç–µ JSON, –º—ã –º–æ–∂–µ–º —Å—Ä–∞–∑—É –ø–æ–ª—É—á–∏—Ç—å –∏—Ö –≤ –≤–∏–¥–µ —Å–ª–æ–≤–∞—Ä—è Python.

### GET —Å –ø–∞—Ä–∞–º–µ—Ç—Ä–∞–º–∏

**`chunked(iterable, n)` ‚Äî —Ä–∞–∑–±–∏–≤–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –Ω–∞ —á–∞–Ω–∫–∏ (—á–∞—Å—Ç–∏) –ø–æ n —ç–ª–µ–º–µ–Ω—Ç–æ–≤**

**windowed(iterable, n) ‚Äî —Å–æ–∑–¥–∞—ë—Ç "—Å–∫–æ–ª—å–∑—è—â–µ–µ –æ–∫–Ω–æ" –ø–æ n —ç–ª–µ–º–µ–Ω—Ç–æ–≤**

**collapse() ‚Äî —Ä–∞—Å–ø–ª—é—â–∏–≤–∞–µ—Ç –≤–ª–æ–∂–µ–Ω–Ω—ã–µ —Å–ø–∏—Å–∫–∏**

**`partition(predicate, iterable)` ‚Äî —Ä–∞–∑–±–∏–≤–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –Ω–∞ –¥–≤–∞: –ø—Ä–æ—à–µ–¥—à–∏–µ –∏ –Ω–µ –ø—Ä–æ—à–µ–¥—à–∏–µ —É—Å–ª–æ–≤–∏–µ**
